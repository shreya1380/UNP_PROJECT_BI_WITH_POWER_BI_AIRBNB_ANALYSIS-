# UNP_PROJECT_BI_WITH_POWER_BI_AIRBNB_ANALYSIS

I did this project in collaboration with UNP, where I developed an Airbnb data analysis dashboard using Power BI. The project analyzes key aspects such as listings, pricing, revenue, reviews, and availability, and uses interactive visualizations to highlight trends, high-performing areas, and key revenue drivers, helping in better decision-making. Additionally, the dashboard includes filters and dynamic visuals that allow users to explore data based on different locations and categories, and this project helped me gain hands-on experience in data visualization, data cleaning, and transforming raw data into meaningful business insights.
